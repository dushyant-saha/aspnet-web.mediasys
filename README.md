# aspnet-web.mediasys
Task application for mediasys
Task Manager application
Important
    • Do not use any database
    • Store all data in backend memory only
    • Data loss after server restart is acceptable
Features
    1. Login
        ◦ Use hardcoded users in backend memory
        ◦ Return a token/session on successful login
    2. Task CRUD
        ◦ Create task
        ◦ List tasks
        ◦ Update task
        ◦ Delete task
        ◦ Each user should see only their own tasks
    3. Task fields
        ◦ title
        ◦ description
        ◦ status
        ◦ priority
        ◦ createdAt
    4. Frontend
        ◦ Login page
        ◦ Task dashboard
        ◦ Add/edit/delete task
        ◦ Search by title
        ◦ Filter by status and priority
        ◦ Show loading and error states
    5. Deliverables
